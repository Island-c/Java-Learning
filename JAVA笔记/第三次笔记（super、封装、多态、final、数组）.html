super方法

super是直接父类的引用，使用super访问父类中被子类覆盖的方法和属性super.f(); super.value=10;

如果构造方法第一行代码没有super（）或者this()，java会默认调用super（），调用父类的无参数构造方法，super（）可以省略。（含义是 ：构造子类时都要先构造父类）

封装Encapsulation

高内聚、低耦合

修饰符同一个类同一个包子类所有类
private*
default**
protected***
public****

原则：
类的属性，使用private权限
set,get方法（eclipse可以自动生成），使用public
类方法一般public

多态（polymorphism）

  1. 多态是方法的多态，不是属性的多态
  2. 三个必要条件：继承、重写、父类引用子类对象
  3. 使用父类引用子类对象，调用子类重写的方法时，就出现了多态。

向上转型和向下转型

自动向上转型：子类转换成父类Animal d = new Dog();

强制向下转型：Dog d2=(Dog)D;
父类引用子类对象时，不能使用子类独有的方法，必须转型成子类引用子类对象。

异常：ClassCastException。

final关键字

修饰变量： 变量为常量，只能复制一次
修饰方法：可以被重载，不能被重写
修饰类：该类不能被继承（String类就是final的）

数组

  1. 声明数组并没有实例化任何对象，只有实例化数组对象时，JVM才在堆上分配空间。
  2. 声明数组时数组并没有被创建。
  3. 构造数组必须指定长度。

int[] arr_1=new int[10]; //默认全是0

for-each循环

仅用于读取！！！

for(type name : 数组)
{
//仅读取，不能修改元素的值
    System.out.println(name);
}
